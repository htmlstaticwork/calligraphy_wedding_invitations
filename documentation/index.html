Introduction

Welcome to the Eleganza Calligraphy & Wedding Invitations website documentation. This guide is designed to help you set up, customize, and maintain your premium calligraphy invitation website efficiently.

Project Goal: To provide a sophisticated digital presence for artisanal calligraphy studios, focused on luxury aesthetics and high-end client conversion.

Installation Guide

1. Local Setup

Running the template locally for development is straightforward since it uses standard HTML5/CSS3/JS.

  1. Extract the .zip package to your desired location.
  2. Open the folder in a code editor (VS Code recommended).
  3. Right-click index.html and select "Open with Live Server" or simply double-click the file to view it in your browser.

2. Hosting & Deployment

To deploy the site to a live server, follow these hosting options:

Option Difficulty Action
GitHub Pages Easy Push folder to GitHub repo and enable Pages in settings.
Netlify / Vercel Easy Drag and drop the folder into their dashboard.
CPanel / FTP Moderate Upload all files to the public_html directory via FileZilla.

Customization Guide

1. Branding & Colors

Change global colors in the CSS variables block found at the top of css/style.css (or the main style sheet).

/* TODO: Replace brand colors below */
:root {
    --primary-color: #0f1113; /* Background */

    --text-color: #ffffff;    /* Primary Text */
}

2. Typography

The template uses Google Fonts. To change them, replace the links in the <head> of all HTML files:

<!-- TODO: Replace Google Font Link -->
<link href="https://fonts.googleapis.com/css2?family=YOUR_FONT" rel="stylesheet">

3. Content Replacement

All text areas are marked with <!-- Content --> tags. To swap images, place your new images in the /images folder and update the img src attributes.
